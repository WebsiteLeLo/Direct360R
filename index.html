You need two things: a YouTube API Key (to load videos & comments) and a Google OAuth Client ID (to post real comments while logged in).

① YouTube Data API Key
  1. Go to Google Cloud Console → create/select a project
  2. Enable YouTube Data API v3
  3. Go to Credentials → Create API Key
② OAuth 2.0 Client ID (for posting comments)
  1. In the same project → Credentials → Create OAuth 2.0 Client ID
  2. Application type: Web application
  3. Authorised JavaScript origins: add http://localhost:8080 and null (for file://)
  4. Copy the Client ID (ends in .apps.googleusercontent.com)
  5. Also enable YouTube Data API v3 scope in OAuth consent screen
⚠️ Important: To post real comments, serve this file via a local server (python -m http.server 8080) and open http://localhost:8080. Google OAuth doesn't work on file:// for security reasons. Video playback works on file:// but comment posting needs localhost.
